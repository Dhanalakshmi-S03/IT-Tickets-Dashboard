PROJECT TITLE:
IT Tickets Analysis Dashboard – Power BI
PROJECT OVERVIEW:
This dashboard provides a complete view of IT help desk performance by tracking ticket volume, severity, priority, gender distribution, and department-level workload. It helps IT teams monitor patterns, identify bottlenecks, and make informed decisions to improve service efficiency.
KEY METRICS:
•	Total Users: 809
•	Total Tickets: 23K+
•	Total Departments: 15
•	Total Days Analyzed: 12K
•	Last Refresh Date: 07-08-2025
DASHBOARD FEATURES:
1.	Ticket Distribution by Severity:
Visual donut chart showing Minor, Major, and Critical ticket proportions.
2.	Tickets by Priority:
Breakdown of High, Medium, and Low priority requests.
3.	Tickets by Type and Gender:
Helps understand user demographics and request categories.
4.	Business Unit & Group-Level Ticketing:
Bar chart showing ticket counts across branches like New York, Houston, Seattle, and Los Angeles.
5.	Department-Level Ticket Analysis:
Visual comparison of ticket volume across Accounting, Tech Support, HR, Payroll, Customer Service, Quality Assurance, etc.
6.	Severity Filters:
Multiple slicers allow filtering by severity, department, business unit, and gender.
TOOLS USED:
•	Power BI Desktop
•	Power Query for cleaning help desk logs
•	DAX for calculating response days and aggregation metrics
